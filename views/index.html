Image Search Abstraction Layer

by Sam

Endpoint: https://sam-img-search.glitch.me/

Input a search term. The service outputs some images and metadata. History can also be fetched.

Code Samples:

Search for images. 'offset' parameter provides pagination. 

https://sam-img-search.glitch.me/cats/offset=2
The above would return an array of 10 objects with the format:
